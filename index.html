Connexion Google Sheets

Le CRM lit et écrit dans un Google Sheet partagé. Toute votre équipe voit les mêmes données. Le Google Sheet reste modifiable directement comme un Excel en ligne.

Guide de configuration (5 minutes)

1. Créez un Google Sheet avec 3 onglets : Entreprises, Echanges, Equipe
2. Ajoutez les en-têtes en ligne 1 (voir le guide Word)
3. Partagez le Sheet en "Tout le monde avec le lien peut modifier"
4. Allez sur console.cloud.google.com > créez un projet > activez "Google Sheets API" > créez une clé API
5. Dans le Google Sheet, menu Extensions > Apps Script, collez le script fourni dans le guide, déployez en Web App
6. Collez les 3 infos ci-dessus et cliquez "Activer"
